== Description ==

Changes the quantity field on your product page into a dropdown. Useful when you only have limited stock.

The minimum and maximum values of the quantity field are the minimum and maximum values available from the dropdown.

== Installation ==

1. Install and activate the plugin through the 'Plugins' menu in WordPress.
2. On products for which you manage stock, the quantity field is changed to a dropdown (no need to edit any settings).
